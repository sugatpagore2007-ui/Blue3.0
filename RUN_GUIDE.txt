BLUE NATURAL AI - RUN GUIDE

1. Install requirements:
   pip install -r requirements.txt

2. Optional local brain:
   Install Ollama, then run:
   ollama run llama3.2:3b

3. Start all modules together:
   python run_all.py

4. Voice controls:
   mute blue      -> Blue becomes silent and ignores commands
   unmute blue    -> Blue listens/responds again
   stop blue      -> Blue shuts down safely

5. Safety:
   Risky actions ask confirmation.
   Dangerous terminal/destructive commands are blocked.
   Passwords/OTP/PIN/API keys are not handled by Blue.
